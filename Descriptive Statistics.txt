import pandas as pd
import matplotlib.pyplot as plt

# Load Iris dataset
iris = pd.read_csv(r"C:\Users\HP\Downloads\iris.csv")

# Display dataset preview
print("Iris Dataset Preview:")
print(iris.head())

# Group dataset by species
grouped = iris.groupby('species')

# Display statistics for each species
for name, group in grouped:
    print("\nSpecies:", name)
    print(group.describe())

# Create boxplot
iris.boxplot(by='species', figsize=(8, 6))

# Add title
plt.title("Iris Dataset Boxplot")
plt.suptitle("")

# Display plot
plt.show()
plt.show() 
iris.hist(figsize=(8,6)) 
plt.suptitle("Distribution of Iris Features") 
plt.show()

# Boxplot Explaination:
The dataset contains:

Sepal Length
Sepal Width
Petal Length
Petal Width

Species:

Setosa
Versicolor
Virginica
1. Boxplot Graph Explanation

The first image shows boxplots for iris features.

What Boxplot Shows

A boxplot displays:

Median (middle line)
Spread of data
Minimum and maximum values
Outliers

A) Petal Length Boxplot

Setosa has very small petal lengths.
Versicolor has medium petal lengths.
Virginica has largest petal lengths.

Meaning
Petal length clearly separates the species.
Virginica flowers are largest.

Outliers
Small circles outside box indicate unusual values.

B) Petal Width Boxplot

Setosa has very small petal width.
Virginica has largest petal width.

Meaning
Petal width is useful for classification of species.

C) Sepal Length Boxplot

Virginica has highest sepal length.
Setosa has smallest sepal length.

Meaning
Sepal length varies across species but overlap exists.

D) Sepal Width Boxplot

Setosa has wider sepals.
Versicolor has smaller spread.

Meaning
Sepal width differences are less clear compared to petal features.

Histogram Graph Explanation :
